A plugin add tab to bottom of vedio for create Qrcode-embed and share buttons 

== Description ==

A plugin add tab to bottom of vedio for create Qrcode-embed and share buttons 

by this plugin you can: 

 1-add embed to each vedio
 2-add Qrcode to each vedio
 1-add Share button to each vedio



== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
